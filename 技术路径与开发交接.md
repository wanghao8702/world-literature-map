# 世界文学巡礼 · 3D 文学地图 —— 技术路径与开发交接文档

> 一句话定位：一张可交互的 3D 世界地图，以"古地图 / 水墨"美术风格呈现世界名著的地理分布，支持点击看书卡、按地区筛选、镜头飞行巡游、天气特效。灵感来自抖音 @麻省理工Rui同学 的"世界文学巡礼"作品（vibe coding 类）。
>
> 本文档用于交接给 **Claude Code** 继续开发。当前已有一个可运行的脚手架 `literary-map.html`，下文标注了它的现状、可扩展点、迁移到生产栈的路线，以及分阶段任务清单。

---

## 1. 背景与目标

- **来源**：参考了一个抖音爆款作品——一张 3D 文学地图，"跨越 107 国 · 收录 1302 部名著"，镜头在地图上飞行巡游，点击地标弹出书卡，配有雨雪天气和巡游路线。
- **目标产物**：一个可在浏览器运行的单页 Web 应用（最终可托管上线），呈现规模化的名著地理数据（目标量级 1000+ 本），具备良好的美术质感、流畅的 3D 体验和叙事性的镜头/交互。
- **开发模式**：vibe coding —— 核心难点不在渲染（地图引擎已封装好 3D 地形/天气），而在 **① 整理结构化的书籍地理数据** 和 **② 调出独特的美术风格**。

---

## 2. 原型技术栈分析（对原视频的推断）

原作者未公开技术说明，以下是根据画面特征做的判断，作为生产栈的参考方向：

| 特征 | 对应技术 |
| --- | --- |
| 真实地理 + 真 3D 地形（真实海岸线、河流、带阴影的山脉） | **Mapbox GL JS** 的 `setTerrain` + raster-dem 高程源 |
| 天气预设（英伦细雨 / 北欧飘雪等） | Mapbox GL JS **v3.9+** 原生 `map.setRain()` / `map.setSnow()` |
| 自定义地标、抬升符号、光照/雾/景深 | Mapbox v3 Standard 风格的环境效果 + 自定义符号 |
| 水墨/古地图质感 | 自定义 Mapbox Studio style，或地理配准过的手绘/AI 贴图作为 raster 源 |
| 镜头巡游（丝路巡游 / 世界经典巡礼） | `flyTo` / 相机关键帧动画 |
| 1302 本书 + 豆瓣评分 + 国家 | 一份带经纬度的 GeoJSON 数据集驱动标记与书卡 |

**结论**：原型最可能是 **Mapbox GL JS v3**（也可能是其开源分支 MapLibre GL），生产环境推荐对齐 Mapbox v3。

---

## 3. 技术选型决策

| 维度 | 当前脚手架（demo） | 生产目标 |
| --- | --- | --- |
| 地图引擎 | **MapLibre GL JS 4.7.1**（免 token、免注册，打开即跑） | **Mapbox GL JS v3**（原生天气、Standard 风格、官方高程更精） |
| 底图贴图 | Esri World Physical Map（免费栅格，旧纸质感） | 自定义 Mapbox style 或手绘/AI 水墨贴图 |
| 3D 地形 | MapLibre 官方免费 DEM | Mapbox `mapbox.mapbox-terrain-dem-v1` |
| 天气 | canvas 粒子模拟雨/雪 | `setRain()` / `setSnow()` 原生一行调用 |

> **为什么 demo 用 MapLibre**：让交接方无需申请任何密钥即可看到效果，且 API 结构与 Mapbox 几乎一一对应，迁移成本低。MapLibre 也是免费长期方案，若不需要原生天气可一直用它。

## 4. 当前脚手架现状（`literary-map.html`）

单文件，无构建步骤，浏览器直接打开即可运行。

### 已实现功能
- 3D 地形（可拖拽旋转 / 倾斜）+ 旧纸质感底图 + sky/雾兜底
- 12 部示例名著的**印章式标记**，点击 → 镜头飞过去 + 弹出**书卡**（书名 / 原名 / 作者 / 年份 / 豆瓣分 / 国家城市 / 一句话简介）
- 底部**地区筛选**胶囊（全部 / 亚洲 / 欧洲 / 美洲 / 中东·非洲）
- **飞行巡游**按钮：依次飞过当前筛选的书目并自动弹卡
- 右上角**天气切换**：晴 / 雨 / 雪
- 顶部标题与"跨越 N 国 · 收录 M 部"统计（由数据自动计算）

### 代码中的关键扩展点（已用 ①②③④ 注释标记）
- **① 换数据**：`books` 数组 —— 替换为真实数据集（建议改为外部 GeoJSON 加载）
- **② 换贴图**：style.sources.physical 这层 —— 替换为自定义水墨贴图 / Mapbox style
- **③ 换地形**：style.sources.terrainSource —— 替换为 Mapbox DEM 源
- **④ 换天气**：canvas 粒子的 `setWeather()` —— 迁移到 Mapbox 后改用原生 `setRain/setSnow`

### 已做的性能优化（重要，避免回退）
- 去除所有 UI 的 `backdrop-filter: blur()`（毛玻璃）—— 在动态 WebGL 上会逼迫每帧重新合成+模糊整屏
- 去除暗角层的 `mix-blend-mode: multiply` —— 同样导致整屏逐帧混合
- `map.setPixelRatio(min(devicePixelRatio, 1.5))` —— 高分屏上避免 3D 地形按 2~3 倍像素渲染（提升最明显）
- 地形 `exaggeration` 1.6 → 1.4；`fadeDuration:0`；收紧 `maxTileCacheSize`；雨雪粒子减量 + 后台暂停

## 5. 数据模型

当前示例对象结构（字段已够用，可直接扩展）：

```js
{
  t:"我与地坛",            // 书名（中文）
  en:"Me and the Temple of Earth", // 原名 / 英文名
  a:"史铁生",              // 作者
  y:2011,                  // 出版/成书年份
  country:"中国", city:"北京",
  coord:[116.41, 39.90],   // [经度, 纬度]
  r:9.5,                   // 豆瓣评分
  region:"亚洲",           // 用于筛选/分组的地区标签
  d:"……"                  // 一句话简介（注意版权：用自己的话，勿抄原文/书摘）
}
```

**规模化建议（1302 本）**：拆成外部 `books.geojson`（FeatureCollection），运行时 `fetch` 加载并懒加载详情；标记必须上**聚合 clustering**（MapLibre/Mapbox 内置 cluster 或 supercluster），否则一屏几百个 marker 会严重卡顿。可补充字段：封面图 URL、世纪/年代分组、主题标签、名句、外链（豆瓣/购买/试读）。

## 6. 迁移到 Mapbox GL JS v3 的步骤

1. 头部脚本换成 mapbox-gl 的 JS/CSS，设置 `mapboxgl.accessToken = '...'`（在 account.mapbox.com 免费申请，额度宽裕）。
2. style 换成 Mapbox Standard 或自定义 style（Mapbox Studio 里做水墨风，或叠一张地理配准的栅格贴图）。
3. 地形源换成 `mapbox-dem`（`mapbox://mapbox.mapbox-terrain-dem-v1`），`setTerrain` 用法不变。
4. 天气：删掉 canvas 粒子，改用原生 `map.setRain({...})` / `map.setSnow({...})`。
5. 其余 `flyTo`、`Marker`、巡游逻辑、筛选、书卡几乎原样可用。

> 注意：`localStorage` 在下载到本地直接打开的文件 / 正式托管环境里可用；但在 claude.ai 聊天内嵌预览的沙箱里被禁用。做"想读/已读"等持久化时以本地或托管环境为准。

## 7. 功能路线图（按价值/还原度排序）

### P0 —— 最能还原原片质感、优先做
- **换水墨/AI 贴图**：替换底图风格，是与原片差距最大的一处。
- **时间轴展开 + 弧线连接**：年份滑块让标记按世纪逐渐浮现；城市/作家间拉流动虚线（丝路、作家足迹、影响关系）。
- **故事/章节模式**：像翻书一样带旁白文字、镜头自动逐站讲解 —— 还原原视频的"解说"灵魂。

### P1 —— 信息量与氛围
- 古地图装饰：罗盘玫瑰、做旧边框、海洋帆船/海怪、经纬网格线。
- 会动的装饰元素：沿路径运动的骆驼 / 帆船 / 飞鸟（symbol 动画）。
- 晨昏 / 季节光照（按真实时间调色 + 方向光）、山间云雾。
- 数据可视化面板：各大洲 / 世纪 / 评分分布的小图表；国家按收录量热力染色。

### P2 —— 交互与多媒体
- 顶部搜索框（书名/作者/国家，飞过去）；URL hash 深链分享。
- "想读/已读"书单（见第 6 节 localStorage 注意事项）。
- 用户自定义巡游路线；点击区域看该地代表作。
- 声音：背景古琴 / 翻书音效 / 与天气联动的环境音；可选每本书朗读片段。
- 多套皮肤一键切换（水墨 / 羊皮卷 / 星空夜）。

### 工程性（贯穿各阶段）
- 1302 本上聚合 clustering + 外部 GeoJSON 懒加载。
- 开场动画（太空俯冲落地）、idle 自动巡航、相机用平滑路径曲线替代简单 `flyTo`。
- 加一个"性能模式"开关：一键在「3D 地形」与「流畅平面」间切换，照顾弱机。

## 8. 给 Claude Code 的可执行任务清单

按建议顺序，每条都是一个相对独立、可验证的小任务：

- [ ] **数据层重构**：把内联 `books` 数组抽成外部 `books.geojson`，运行时 fetch 加载；补充 cover/世纪/主题/名句/外链字段。
- [ ] **聚合 clustering**：标记改用 cluster 源，缩放时自动聚合/展开，点簇可展开或放大。
- [ ] **换美术风格**：接入水墨/AI 贴图或自定义 style（先做一个可切换的"皮肤"机制，便于试不同贴图）。
- [ ] **时间轴组件**：底部年份滑块，过滤/渐显标记，可播放"随时间展开"动画。
- [ ] **弧线连接图层**：在指定书目/城市之间绘制流动虚线（great-circle arc + 动态 dash）。
- [ ] **故事/章节模式**：定义一组"章节"（每章 = 一段旁白 + 一条相机路径 + 高亮的几本书），做上一/下一步导航与自动播放。
- [ ] **性能模式开关**：右上角加按钮，切换 `setTerrain(null)` ↔ 开启地形。
- [ ] **搜索 + 深链**：搜索框 + URL hash（如 `#book=...`）打开自动定位。
- [ ] **（迁移可选）Mapbox 版**：按第 6 节迁移，启用原生 `setRain/setSnow`。

## 9. 交流要点速记（关键决策与结论）

- 原型最可能基于 **Mapbox GL JS v3**；其天气预设对应原生 `setRain()/setSnow()`（v3.9+），3D 地形对应 `setTerrain` + raster-dem。
- demo 选 **MapLibre GL** 是为了"免 token、打开即跑"，且与 Mapbox API 结构对应、迁移成本低；MapLibre 也是可长期使用的免费方案。
- 卡顿主因经定位为三点并已修复：**毛玻璃 `backdrop-filter`**、**`mix-blend-mode`**、**未限制的高分屏渲染像素比**。后续开发务必不要把这三类写法加回到动态地图上。
- 规模化（1302 本）的两条硬性约束：**外部 GeoJSON 懒加载** + **标记聚合 clustering**。
- 美术贴图是"还原原片质感"的最大杠杆；故事章节模式是"还原视频叙事感"的最大杠杆——两者优先级最高。
- 持久化（书单等）注意：`localStorage` 在本地/托管环境可用，在聊天内嵌预览沙箱里不可用。
