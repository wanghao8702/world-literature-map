# 世界文学巡礼 · 3D 文学地图

一张可交互的 3D 世界地图，以古地图质感呈现世界名著的地理分布：点击地标看书卡（封面 / 简介 / 精彩段落）、按地区筛选、镜头飞行巡游、章节解说、城市连线、天气特效。

> 灵感来自一个"世界文学巡礼"的短视频作品。本仓库为可运行的单页 Web 应用（无构建步骤）。

## 运行

浏览器直接打开 `literary-map.html` 即可；若因 CORS 无法读取本地 JSON，用本地静态服务器：

```bash
python -m http.server 8753
# 然后访问 http://localhost:8753/literary-map.html
```

## 技术栈

- **MapLibre GL JS 4.7.1**（免 token）+ Esri World Physical 栅格底图 + 3D 地形（DEM，可在网络可达时开启）
- 单文件 `literary-map.html`，纯前端，无构建

## 数据结构

| 文件 | 作用 |
| --- | --- |
| `books.geojson` | 地理索引层：坐标 + 书名/作者/年份/国家/地区/评分/主题/巡游顺序 |
| `books-content.json` | 富内容层（按 `id` 关联）：简介 / 精彩段落 / 封面路径 / 外链 |
| `covers/` | 本地封面图（`covers/{id}.jpg`），缺图时书卡自动回退为文字封面 |

当前收录 **30 部名著，覆盖 28 个国家**。

## 主要功能

- 标记聚合（clustering），支撑 1000+ 规模
- 浮动书卡：封面图 + 文字封面兜底、豆瓣评分、一句话简介、精彩段落
- 底部地区筛选 + 飞行巡游
- 故事 / 章节解说模式（旁白 + 镜头 + 高亮）
- 城市连线（大圆弧 + 流动虚线）
- 搜索（书名/作者/国家）+ 深链分享（`#book=id`）
- 性能模式开关（3D 地形 ↔ 平面）、罗盘玫瑰
- 天气特效（晴 / 雨 / 雪）
